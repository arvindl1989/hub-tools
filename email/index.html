Connect Your Ticket Sheet

The Email Tracker pulls live data from a Google Sheet you maintain. Set it up once — from then on it loads automatically every time you open the tool.

How to set up your Google Sheet:
  1. Open your ticket spreadsheet in Google Sheets
  2. Make sure columns include: Number, Short description, Sub-Category, Requested by, Assigned to, State, Created
  3. Click File → Share → Publish to web
  4. Select the sheet tab, choose Comma-separated values (.csv), click Publish
  5. Copy the URL provided and paste it below
